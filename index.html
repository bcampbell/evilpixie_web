Evilpixie

Remember when drawing stuff was fun?

Modern paint programs are awesome and everything, but, well... something has gone missing over the years.

Step back into an age when throwing pixels about the place was really _fun_.

Features

  • Animation
  • Deluxe Paint keyboard shortcuts
  • Paint with left button, Erase with right
  • Load/save png, gif, ilbm/iff, tga, pcx, bmp, lots more...
  • Cross platform
  • Quick and snappy-feeling
  • Up to 256 glorious colours!
  • Free!

Development

Evilpixie is Free Software, licensed under GPL v3

Source code is on github: http://github.com/bcampbell/evilpixie (The latest work is on the rgba branch)

You can report bugs and feature requests on the github issue tracker

It is written in C++ and uses QT for gui code. Primary development system is Linux, but it compiles fine on Windows (using Mingw at least), and it'll probably compile fine on OSX with few or no changes, if anyone wants to give it a go.

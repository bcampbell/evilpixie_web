Evilpixie

Remember when drawing stuff was fun?

Modern paint programs are awesome and everything, but, well... something has gone missing over the years.

Step back into an age when throwing pixels about the place was really _fun_.

Features

  • Animation
  • Deluxe Paint keyboard shortcuts
  • Paint with left button, Erase with right
  • Supports png, gif, ilbm/iff, tga, pcx, bmp
  • Cross platform
  • Quick and snappy-feeling
  • Free!

Download

The latest version with windows and mac binaries is 0.2:

https://github.com/bcampbell/evilpixie/releases/tag/v0.2

But really, this is a long way behind the development version. A new release will appear at some point, but for now, you'll get the best results by compiling from source.

Development

Evilpixie is Free Software, licensed under GPL v3

Source code is on github: http://github.com/bcampbell/evilpixie.

You can report bugs and feature requests on the github issue tracker

It is written in C++ and uses QT for gui code. Primary development system is Linux, but it compiles fine on Windows (using Mingw at least), and it'll probably still compile fine on OSX with few or no changes, if anyone wants to give it a go.
